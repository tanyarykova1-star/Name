# 🐰🦔 Smeshariki Guessing Game (School Project Template)

A simple and interactive logic game featuring Krosh and Yoshi (Jozhik), designed as a template for school programming assignments.

## 🎓 Why this project?
I created this as a school project to demonstrate basic Python concepts. It’s a perfect starting point for beginners who want to see how a simple choice-based game works.

## 🛠️ Features
* **Character Logic**: Interactive dialogue with Krosh and Yoshi.
* **Educational Template**: Clean code structure, easy to modify for any other characters.
* **Beginner Friendly**: Uses basic inputs, strings, and conditional statements (`if/else`).

## 🚀 How to use it as a template
1. Copy the code.
2. Replace "Krosh" and "Yoshi" with your favorite characters.
3. Change the questions and answers in the script.
4. Your own school project is ready!
